# Data Science Final Project: Sports Injury Analysis

## Overview
This repository contains a data science final project focused on analyzing injury data across major professional sports leagues, including MLB, NBA, NFL, and NHL. The goal of the project is to clean, process, and analyze the injury datasets to uncover trends and insights.

## Project Structure

- `datasets/`: Contains all data files used in the project.
  - `raw/`: Original, unmodified datasets.
  - `processed/`: Cleaned and prepared datasets ready for analysis.
- `images/`: Generated plots, graphs, and visual assets.
- `notebooks/`: Jupyter notebooks containing exploratory data analysis (EDA), data preparation, and modeling.
  - `01_data_preparation.ipynb`: Initial notebook for cleaning and standardizing the raw data.
- `src/`: Reusable Python scripts and modules used across notebooks.

## Data Sources
The project utilizes historical injury data collected for:
- MLB injuries
- NBA injuries (2010-2020)
- NFL injuries (2009-2021)
- NHL injuries

## Getting Started
To reproduce the analysis or explore the data:
1. Ensure you have a standard data science Python environment installed (e.g., Anaconda or pip-managed environment with `pandas`, `jupyter`, `matplotlib`, `seaborn`).
2. Open the `notebooks/` directory and run the notebooks in sequence.
